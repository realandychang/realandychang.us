My Story

  I am really into technology since I was around 5, this could be shown by whenever my teacher wants me to do some Powerpoint presentations, etc. I could do well on those.

  The first time I made my website was in 2019 using Google Site. At that period, I used to think that to build a website, all I need to do is drag and type.

  After a while, I found out that the function of Google Site is limited, so I intend to find another platform to build my website that has more features than Google Site. Then, I found Wix, a website builder that contains a lot of cool features, but after a period of using Wix. I found out that the website build using Wix isn't responsive and hard to manage, so I started my actual scripting period or career.

  In the year 2021, I found Github Pages. Till now, I am still using Github Pages to edit my website.
